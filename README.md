# じゃんけんゲーム
## これは何？
[GitHub Copilot](https://github.com/features/copilot/)を使ってGoで作ったじゃんけんゲーム

## 使い方
```bash
git clone git@github.com:askdkc/janken-go.git
cd janken-go
go run .
```
